# Extractor Highlights Kindle
Programita de HTML para extraer los highlights de Kindle desde la web.

<h2>Funcionamiento:</h2>
<ol><li>Accedes a la web de notas y destacados de Kindle: <a href="https://read.amazon.com/notebook">https://read.amazon.com/notebook</a></li>
<li>Entra en el libro que quieras extras todas tus notas y destacados y abre la función de inspeccionar con F12.</li>
<li>Copia todo el código del body e introdúcelo en el programa.</li></ol>
